# 🍅 番茄时钟 Windows 桌面小组件

[**English**](./README.md)

一个基于 Electron 构建的轻量级、高颜值、支持沉浸式体验的 Windows 桌面番茄时钟。它模仿了 macOS 的组件体验，以无边框、冷色调毛玻璃的悬浮窗口独立运行在任务首层。

## ✨ 核心特性

- **极致沉浸的外观**：半透明毛玻璃材质 (Glassmorphism)，全平台无边框融合。
- **真正的原生桌面组件**：支持全局置顶不被遮挡，通过拖拽标题栏可放置在屏幕任意边缘。
- **平滑自适应的动画**：内置 SVG 同步衰减环形计时器，多状态自适应冷色调主题 (番茄红/青薄荷/天空蓝)。
- **免媒体文件的本地铃声**：使用浏览器底层的 Web Audio API 合成轻音，不依赖外部音频文件，体积更小。
- **系统原生通知**：工作结束、休息开始与结束，无缝触发 Windows 系统右下角消息提醒。
- **智能工作流模式**：
  - 自动管理：“番茄 25 分钟 -> 短休 5 分钟”的闭环循环。
  - 成就激励：每连续完成 4 个番茄将自动建议进入 15 分钟长休模式。
  - 数据留存：跨天自动重置的今日专注数据记录，历史偏好设置永久保存。

## 🚀 快速启动

你需要安装 `Node.js` (推荐 v18+)。目前已验证在 v22 环境下完美运行。

1. **克隆/进入项目目录**
   ```bash
   cd Tomato-Timer
   ```

2. **安装依赖文件**
   ```bash
   npm install
   ```

3. **运行程序**
   ```bash
   npm start
   ```

## 🛠️ 将项目打包为免安装的 .exe (可选)

如果你想把这个有趣的工具发给没有开发环境的朋友使用，可以通过 `electron-builder` 将其打包为 `.exe`：

1. **安装本地打包器**：
   ```bash
   npm install electron-builder --save-dev
   ```
2. **在 `package.json` 内配置构建选项（已准备）然后执行**：
   ```bash
   npx electron-builder --win
   ```

## 👨‍💻 技术栈

- 运行容器：`Electron`
- 界面搭建：原生 `HTML5` + `Vanilla CSS3` (无框架、重设计)
- 全局逻辑：原生 `JavaScript` 工具集 (`localStorage` / `Web Audio API`)
